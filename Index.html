📅 How to Import the Calendar

Google Calendar:
1. Download the .ics file
2. Open Google Calendar on desktop
3. Click ⚙️ Settings → Import & Export
4. Click "Select file" → choose the .ics file
5. Choose your calendar → Import ✅

Microsoft Outlook:
1. Download the .ics file
2. Open Outlook → File → Open & Export
3. Click "Import/Export" → Import an iCalendar
4. Select the .ics file → Open ✅

iPhone / Android:
1. Download the .ics file to your phone
2. Tap the file — it opens your calendar app automatically
3. Tap "Add All" to import all reminders ✅
